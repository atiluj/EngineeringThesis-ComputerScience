## Zadanie 1
Dana jest lista [15, 10, 5, 1, 21, 12, 9]. Napisz program, który wypisze:
* najmniejszy element tej listy
* największy element tej listy
* średnią liczb tej listy
* listę posortowaną rosnąco 
* listę posortowaną malejąco

<Dropdown title="Rozwiązanie">
  <Code>
    lista = [15, 10, 5, 1, 21, 12, 9] <br/>
    print("min:", min(lista)) <br/>
    print("max:", max(lista)) <br/>
    print("średnia:", sum(lista) / len(lista)) <br/>
    lista.sort() <br/>
    print("posortowana rosnąco:", lista) <br/>
    lista.sort(reverse=True) <br/>
    print("posortowana malejąco:", lista) <br/>
  </Code>
