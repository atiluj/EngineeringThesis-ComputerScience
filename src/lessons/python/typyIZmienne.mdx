## Python jako kalkulator
Uruchom IDLE shell. Jeśli z niego nie korzystasz, to włącz terminal i uruchom pythona w konsoli wpisując: 

```bash
python3
```

Możemy używać teraz Pythona tak samo jakbyśmy używali kalkulator:

```python
>>> 2 + 5
7
>>> 2 * 9 - 15.5
2.5
>>> 2 * (9 - 15.5)
-13.0
>>> 2 * 2 + 2
6
>>> 2 + 2 * 2 
6
```
Zwróć uwagę na to, że Python zna kolejność wykonywania działań (mnożenie wykonuje się przed dodawaniem), więc nie musisz pisać zbędnych nawiasów. 

## Operatory
W Pythonie wyróżniamy następujące operatory: <br/>
`+` dodawanie <br/>
`-` odejmowanie <br/>
`*` mnożenie <br/>
`/` dzielenie <br/>
`//` dzielenie całkowite <br/>
`**` potęgowanie <br/>
`%` modulo (reszta z dzielenia) <br/>

```python = 
>>> 9 / 2
4.5
>>> 9 // 2 
4
>>> 2 ** 3 
8
>>> 9 % 2
1
```

## Typy
W Pythonie wyróżniamy cztery podstawowe (proste) typy danych:
* `int` - liczby całkowite
* `float` - liczby zmiennoprzecinkowe
* `bool` - typ logiczny - prawda/fałsz
* `string` - napisy

#### Typy liczbowe

W podtematach "Python jako kalkulator" oraz "Operatory" wykonywaliśmy operacje na dwóch typach liczbowych - `int` oraz `float`. 

Float'y, czyli liczby zmiennoprzecinkowe zapisujemy z kropką, a nie z przecinkiem!

```python
>>> 2.0
2.0 
>>> 4.5
4.5
>>> 4,5
(4,5) 
# w ten sposób utworzyliśmy krotkę
# więcej o nich dowiemy się w następnym wykładzie!
```

Warto zwrócić uwagę na to, że mimo iż są to dwa różne typy, to możemy wykonać jakieś działanie arytmetyczne, np. dodać floata do inta bez potrzeby konwersji z jednego typu na drugi. Python robi to za nas. 

```python
>>> 4 + 4.5 
8.5
```

Wynik dowolnego działania zawierającego floata i inta jest typu float. 

Co ciekawe wynikiem dzielenia (`/`) dwóch liczb jest również float (nawet jeśli dzielimy dwa int'y, a wynik nie ma części dziesiętnej)

```python
>>> 8 / 2
4.0 
```

#### Bool
Zmienna boolowska może przyjąć  jedną z dwóch wartości logicznych - True (prawda)  lub False (fałsz). Zwróć uwagę na to, że zapisujemy je z dużych liter.

```python
>>> True
True
>>> False
False
```
Co się stanie kiedy spróbujemy użyć jednego z operatorów arytmetycznych na zmiennych boolowskich? 
```python
>>> True * True
1 
>>> True * False
0
>>> True + True
2
>>> False - True
-1
```
Dzieje się tak dlatego, że pod spodem True kryje się tak naprawdę 1, a pod False 0. 

#### String
Stringi, czyli napisy zapisujemy za pomocą cudzysłowiu `""` lub za pomocą apostrofów `''`. 

```python
>>> 'Ala ma kota'
'Ala ma kota'
>>> "Ala ma kota"
"Ala ma kota"
>>> "Ala ma kota'
SyntaxError: unterminated string literal (detected at line 1)
>>> Ala ma kota
SyntaxError: invalid syntax
```
Nie ma znaczenie czy używamy cudzysłowiu czy apostrofów - ważne jest jednak, żeby każdy string kończył się tym samym znakiem, od którego się zaczął - w przeciwnym przypadku interpreter zwróci błąd. Radzimy również, aby w całym programie używać albo tylko cudzysłowiu albo tylko apostrofów - zwiększy to czytelność kodu.


Na stringach możemy wykonać dwa rodzaje działań: 
* dodawanie dwóch napisów (konkatenacja)
* mnożenie napisu przez liczbę

```python
>>> 'Ala ma ' + 'kota'
'Ala ma kota'
>>> 'Ala' * 5
'AlaAlaAlaAlaAla'
```

A co w przypadku, kiedy chcemy dodać liczbę do napisu? 

```python
>>> 'Mam ' + '18' + ' lat'
'Mam 18 lat'
>>> 'Mam' + 18 + 'lat'
Traceback (most recent call last):
  File "<pyshell#33>", line 1, in <module>
    'Mam' + 18 + 'lat'
TypeError: can only concatenate str (not "int") to str
```

Otrzymujemy błąd, ponieważ nie jest możliwe bezpośrednie dodanie inta do stringa. 
Najpierw musimy dokonać konwersji, czyli zmienić typ inta. Używamy do tego celu funkcji str().

```python
>>> 'Mam ' + str(18) + ' lat'
'Mam 18 lat'
```

#### Konwersja między typami
Przed chwilą usłyszeliśmy pojęcie funkcji, lecz czym właściwie ona jest?
W skrócie, funkcja jest wydzielonym kawałkiem kodu, który możemy używać wiele razy w dowolnych miejscach programu. Funkcja może przyjmować argumenty i zwracać pewną wartość. 
Np. funkcja **str()**, którą poznaliśmy wcześniej przyjmuje jako argument liczbę, a zwraca napis. Funkcje możemy zdefiniować sami - czym zajmiemy się w dalszym wykładzie - lub skorzystać z wielu funkcji wbudowanych, które są dostępne w języku Python.

Funkcje na których się dzisiaj skupimy służą do konwersji między typami:
* str() - zmienia typ obiektu na string
* int() - zmienia typ obiektu na int
* float() - zmienia typ obiektu na float
```python=
>>> str(18.5)
'18.5'
>>> int('18')
18
>>> int(17.5)
17
>>> float(3)
3.0
>>>float('1')
1.0
```

#### type()
To też jest jedna z funkcji wbudowanych, która służy do sprawdzania typu obiektu.  

```python
>>> type(10)
<class 'int'>
>>> type(10.0)
<class 'float'>
>>> type(True)
<class 'bool'>
>>> type(False)
<class 'bool'>
>>> type("Hello world!")
<class 'str'>
```

## Zmienne
Zmienna to taki obiekt, który przechowuje dane w programie. Możemy sobie wyobrazić, że jest to pudełko, któremu nadajemy *nazwę* i wkładamy do środka pewną *wartość*. 

Zapisujemy to w następujący sposób: 
Najpierw piszemy `nazwę` naszej zmiennej, następnie znak równości `=` (nazywany znakiem przypisania), a następnie `wartość`, którą chcemy przypisać do zmiennej. 
```python
nazwa = wartosc
``` 

Do środka "pudełka", czyli do naszej zmiennej możemy wstawić liczbę całkowitą, zmiennoprzecinkową, napis lub wartość prawda/fałsz - czyli każdy z typów, który już poznaliśmy.  

```python
>>> liczba = 3
>>> pi = 3.14
>>> napis = "To jest napis"
>>> prawda = True
```

Tak jak wskazuje ich nazwa zmiennej, możemy zmieniać wartość przypisaną do zmiennej.

```python
>>> x = 3 
>>> x
3
>>> x = 'Ala ma kota'
>>> x 
'Ala ma kota'
```

Wartości zmiennych możemy przypisywać również w jednej linii: 

```python
>>> a, b, c = 1, True, 'Ala ma kota'
>>> b
True
```


Na zmiennych możemy wykonywać dowolne operacje:
```python
>>> x = 10
>>> y = 12
>>> z = x + y - 3
>>> z
19
```
W pythonie podczas tworzenia zmiennej należy od razu przypisać jej wartość. Zmienna nie może istnieć bez wartości! 

```python
>>> a
Traceback (most recent call last):
  File "<pyshell#8>", line 1, in <module>
    a
NameError: name 'a' is not defined

```

## Niepoprawne nazwy zmiennych
Nazwa jaką nadajemy zmiennej, może być praktycznie dowolna - oprócz kilku wyjątków. 

* Nazwa zmiennej musi być jednym ciągiem znaków (bez spacji)
* W nazwie zmiennej nie mogą się znajdować znaki odpowiadające za operatory (+, -, *, itd.)
Jedyne dozwolone znaki to litery alfabetu (a-z, A-Z), cyfry (0-9), lub podkreślenia (_)
* Nazwa zmiennej nie możemy zaczynać się od cyfry
* Nie używamy słów kluczowych w pythonie ani nazw funkcji pomocniczej. Jeśli przykładowo nadpiszemy wartość funkcji, to nie będziemy mogli jej potem używać.

```python
>>> max(3,4)
4
>>> max = 1
>>> max(3,4)
Traceback (most recent call last):
  File "<pyshell#15>", line 1, in <module>
    max(3,4)
TypeError: 'int' object is not callable
```

Nazwy zmiennych są *case sensitive*. Oznacza to, że wielkość liter użytych w nazwie ma znaczenie.

``` python
>>> A = 1
>>> a = 2
>>> A
1
>>> a 
2
```

Istnieją dwa polecane sposoby zapisywania zmiennych:
* z użyciem podkreślenia
    **moja_zmienna**, **dane_osobowe**
* tzw. camel case, czyli kolejne wyrazy piszemy łącznie, rozpoczynając każdy kolejny od wielkiej litery
    **mojaZmienna**, **daneOsobowe**

## Standardowe wejście / wyjście
### Wyświetlanie napisu na ekranie
Instrukcja **print()** w pythonie służy do wyświetlania tekstu lub innych wartości na ekranie. Używamy tej instrukcji po to, aby pokazać na ekranie dane lub wyświetlić pewne informacje na temat działania programu użytkownikowi.

Aby użyć tej funkcji, podajemy wartość (lub kilka wartości), które chcemy wyświetlić jako argumenty tej funkcji.

```python=
print("Hello world!"); # Wyświetli "Hello world!" na ekranie

a = 10
b = 12

print(a, b) # Wyświetli "10 12"
print(a + b) # Wyświetli "22"
```

## Pobieranie wartości od użytkownika
Do pobierania danych od użytkownika służy funkcja **input()**. Używamy jej, aby zapytać użytkownika o dane, aby następnie wykorzystać je w naszym programie. 

Zapytajmy użytkownika o jego imię i przywitajmy się z nim:
```python=
imie = input("Podaj imię: ")
print("Cześć ", imie) 
```

Następnie spytajmy o jego wiek i wyświetlmy informację o tym w jakim będzie wieku za 5 lat. 
Musimy w tym momencie uważać, bo 
każda wartość jaką pobierzemy od użytkownika jest zapisywana jako string!
```python=
imie = input("Podaj imię: ")
print("Cześć ", imie) 

wiek = input("Podaj swój wiek: ")
wiek = int(wiek) # Zmieniamy typ zmiennej wiek ze stringa na inta
print("Za 5 lat będziesz mieć ", wiek + 5, " lat")
```
Możemy też trochę skrócić nasz program, zmieniając typ zmiennej wiek od razu przy instrukcji **input()**

```python=
imie = input("Podaj imię: ")
print("Cześć ", imie) 

wiek = int(input("Podaj swój wiek: "))
print("Za 5 lat będziesz mieć ", wiek + 5, " lat")
```
